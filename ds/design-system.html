Scriberr UI design system

A Flexoki-inspired interface language adapted for Scriberr's transcription workspace. The system keeps warm paper, disciplined ink, Iosevka typography, and quiet red-orange accents, then replaces heavy borders with translucent glass, blur, inset highlights, and soft elevation.

Reading system

Edited today · 1,842 words · Private notebook

The interface should feel like a sheet of paper with memory. Text stays dark and steady, metadata recedes, and controls become legible only when the writer needs them.

Flexoki's warmth is strongest when it is disciplined: paper for the page, softened neutrals for structure, and saturated accents only for actions or meaning.

Annotation color should carry intent. Blue marks reference, green marks resolution, yellow marks uncertainty, and red marks destructive or irreversible action.

For long sessions, the system avoids sterile white and avoids low-contrast nostalgia. It should read quietly, but still feel crafted.
